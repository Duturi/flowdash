## 1. 프로젝트 개요

본 프로젝트는 요구사항을 기반으로  칸반 구조의 작업 보드로  
기간 필터, 검색/정렬, 통계 및 달성률을 제공하는 태스크 관리 대시보드 구현 과제이다.  
<br>
CRUD, 테마설정 및 UX요소를 포함하여, 팀 단위 협업을 통해 설계 및 구현을 진행했다.
