## 1. 프로젝트 개요

본 프로젝트는 요구사항을 기반으로 칸반 구조의 작업 보드로  
기간 필터, 검색/정렬, 통계 및 달성률을 제공하는 태스크 관리 대시보드 구현 과제이다.

CRUD, 테마설정 및 UX요소를 포함하여, 팀 단위 협업을 통해 설계 및 구현을 진행했다.

## 3. 수행 절차 및 방법

### 3-1. 진행 순서

1. 요구사항 분석 및 전체 레이아웃 설계
2. 보드 구조 및 테마 구현 / 스토리지 값 저장
3. CRUD / 통계 대시 보드 / 상단 영역 구현
4. 반응형 / 초기화 버튼 / UX 구현
5. 필터 / 정렬 / 상태 / 검색 파이프라인 구현

### 3-2. 협업 규칙

- 브랜치 전략: main / dev / feature/\*
- PR 단위: 기능 1개 기준
- PR 전 rebase 필수
- 스펙 변경은 dev merge 전까지만 허용
